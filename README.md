## Run app

-   `yarn install` (For node_modules)

### Ios
-  `cd ios`
-  `rm -rf Pods`
-  `pod cache clean --all`
-  `pod install`
-  `cd ..`
-  `yarn ios`

### Android
-   `yarn android`
